## How To Install

### How To Install NDS UI Kit

1. Install NDS UI Kit ke dalam project anda menggunakan `npm install nds-ui-base-storybook@latest`. Pastikan untuk selalu menggunakan versi terbaru dari package nds-ui-base-storybook.
2. Import CSS di global file project anda `import "nds-base-ui-storybook/dist/style.css"`. Pastikan untuk selalu mengimport css di global agar seluruh file project dapat terintegrasi dengan NDS UI Kit.

### How To Run and Install StoryBook

1. Clone repository [NDS UI Kit](https://bitbucket.bri.co.id/scm/nds/nds-ui-base-storybook.git). lalu lakukan `npm install` pada project untuk menginstall dependencies yang diperlukan.
2. Run aplikasi menggunakan `npm run storybook` Jika belum mempunyai storybook install menggunakan `npx storybook@latest init` pada project anda.


### Development Tools

Environment pengembangan Anda sangat penting. Berikut adalah beberapa alat dan sumber daya untuk membantu Anda menyiapkan ruang kerja Anda:

- **Code Editor**: Kami sangat menyarankan menggunakan [Visual Studio Code](https://code.visualstudio.com/) untuk development. Dapat disesuaikan secara tinggi dan mendukung berbagai ekstensi.

- **Package Manager**: [npm](https://www.npmjs.com/) adalah manajer paket yang kami gunakan untuk proyek JavaScript. Pasanglah untuk mengelola dependensi proyek.
